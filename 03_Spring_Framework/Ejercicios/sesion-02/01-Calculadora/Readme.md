# Creación de una aplicación con Spring y H2.

El ejercicio consiste en implementar la API que has definido para una calculadora anteriormente.
Además, se recomienda hacer uso de estas guías para poder mapear relaciones 1-1, 1-N, M-N con Spring Data JPA (**en el caso de ser necesario**)
- [Guía 1:1](https://www.baeldung.com/jpa-one-to-one)
- [Guía 1:N](https://www.baeldung.com/hibernate-one-to-many)
- [Guía M:N](https://www.baeldung.com/jpa-many-to-many)

## Entrega

La entrega consiste en el código de la implementación (únicamente la carpeta src y archivo pom.xml). Utiliza la base de datos persistente que quieras (MySQL, PostgreSQL) o una base de datos en memoria (H2) para almacenar los datos de la aplicación.
